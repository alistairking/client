Color Command¶

You can toggle the color of the cloudmesh shell console by using the color command.

The manual page of the group command can be found at: group

Color ON/TRUE¶

Turn the color mode ON:

$ cm color ON
  Color True

$ cm color TRUE
  Color True

Color OFF/FALSE¶

Turn the color mode OFF:

$ cm color OFF
  Color False

$ cm color FALSE
  Color False
